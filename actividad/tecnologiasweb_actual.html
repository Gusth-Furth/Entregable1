Tecnologias web actuales Frontend y Backend

  1. GraphQl: Esta herramienta brinda a la sociedad una mejor transferencia de datos y reduce el consumo de ancho de banda.
  2. Lado del cliente: Se maneja JavaScript, Css, HTML brinda dinamismo a los sitios web y es una de las mas utilizadas en la actualidad. En pocas palabras es la interacción entre una aplicación y el usuario.
  3. Lado del servidor: lenguajes de programación como PHP, Java, Ruby, Python, Nodes.js son los que se encargan de la lógica de la aplicación permitiendo que backend realice las solicitudes que envié el frontend
